####################### ТРЕБОВАНИЯ #######################

1. Для запуска необходимо использовать версию Java не ниже 1.8
' java -version
2. В Java, используемую для запуска, предустановлено CryptoPro JCP + TLS
3. Убедиться, что:
* в CryptoPro JCP установлен личный сертификат, который является доверенным на клиентских соединениях
* сертификаты клиентских соединений являются доверенными в CryptoPro JCP

####################### ЗАПУСК ПРОЕКТА #######################

1. Для переопределения настроек запуска необходимо положить рядом с приложением файл application.properties ( пример файла uct-system/uct/src/main/resources/config )
2. Изменить настройки приложения для использования сертификата, установленного в CryptoPro JCP
3. Для запуска выполнить команду
' java -jar uct.jar UctWebApplication
